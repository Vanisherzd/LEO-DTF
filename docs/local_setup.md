# Local Setup Guide

This guide walks through setting up the LEO-DTF repository on a fresh machine
and reproducing the preliminary synthetic evaluation artifacts.

## 1. Clone Repository

```bash
git clone https://github.com/Vanisherzd/LEO-DTF.git
cd LEO-DTF
```

## 2. Create Virtual Environment

```bash
python3 -m venv .venv
source .venv/bin/activate   # Linux/macOS
# .venv\Scripts\activate    # Windows
```

## 3. Install Dependencies

```bash
python -m pip install --upgrade pip
python -m pip install -e .          # install package + dependencies
python -m pip install pytest       # testing
```

If using `uv`:

```bash
uv pip install -e .
uv pip install pytest
```

## 4. Validate Repository State

```bash
python scripts/validate_repo_state.py        # fast checks
# or full validation (slower):
python scripts/validate_repo_state.py --full
```

## 5. Run All Tests

```bash
pytest -q
```

Expected: all tests pass.

## 6. Reproduce Preliminary Artifacts

### 6.1 Generate synthetic pass dataset

```bash
python scripts/generate_synthetic_dataset.py
```

### 6.2 Run Monte Carlo trials

```bash
python scripts/run_monte_carlo_synthetic.py --trials 5
```

### 6.3 Run CRLB sensitivity diagnostic

```bash
python scripts/diagnose_crlb_sensitivity.py
```

### 6.4 Run ambiguity ablation study

```bash
python scripts/run_ambiguity_ablation.py --trials 5 --seed 42
```

### 6.5 Summarize all evaluations

```bash
python scripts/summarize_evaluation.py
```

### 6.6 Generate paper figures

```bash
python scripts/make_paper_figures.py
```

## 7. Expected Output Locations

After running the scripts above, the following artifacts are generated:

| Artifact | Location |
|----------|----------|
| Synthetic pass data | `experiments/results/synthetic_pass_dataset.json`, `synthetic_pass_observations.csv` |
| Monte Carlo results | `experiments/results/monte_carlo_trials.csv`, `monte_carlo_summary.json` |
| CRLB sensitivity | `experiments/results/crlb_sensitivity.csv`, `crlb_sensitivity_summary.json` |
| Ablation summary | `experiments/results/ablation/ambiguity_ablation_summary.json`, `paper/tables/ablation_summary.tex` |
| Evaluation summary | `experiments/results/evaluation_summary.md`, `evaluation_summary.json` |
| Paper figures | `paper/figures/dtf_concept.pdf`, `paper/figures/posterior_heatmap_placeholder.pdf`, etc. |

All output directories are gitignored (see `.gitignore`). Generated artifacts
are local and will not be committed.

## 8. Common Problems

### ModuleNotFoundError

Ensure the virtual environment is activated and the package is installed:

```bash
source .venv/bin/activate
python -m pip install -e .
```

### pytest not found

```bash
python -m pip install pytest
```

### Missing matplotlib

This is a dependency of the package. Reinstall:

```bash
python -m pip install -e .
```

### Generated outputs seem ignored by git

All output directories are intentionally listed in `.gitignore`. This is
normal — do not force-add them. They are local artifacts.

### pdflatex not available

The repository does not require LaTeX to build. The paper figures are
generated by `make_paper_figures.py` using matplotlib. To compile the paper
manually, install a LaTeX distribution (e.g., TeX Live) and run:

```bash
cd paper
pdflatex main.tex
```

### Placeholder citations in paper/refs.bib

The bibliography contains `todo_*` placeholder entries. These must be
replaced with verified citations before manuscript submission. See
`docs/bibliography_todo.md` for the full list.

## 9. Claim Safety Reminder

All results in this repository are **preliminary synthetic diagnostics**.

The following are **explicitly outside scope**:

- No real satellite over-the-air (OTA) validation is claimed
- No meter-level localization accuracy is claimed
- No GNSS replacement is claimed
- No operational deployment is claimed
- LR1121 is used as transmit-only; no LR-FHSS reception is implemented

See `docs/claim_audit_report.md` and `paper/sections/08_limitations.tex`
for the full list of scope constraints.
