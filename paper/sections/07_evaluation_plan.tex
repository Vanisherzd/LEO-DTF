\subsection{Preliminary Synthetic Results (do not constitute paper performance claims)}

The smoke test (20 packets, 600~s synthetic pass, 400~km equatorial orbit,
$\sigma_f=1$ Hz, $\sigma_\tau=1$ ms) produces:
\begin{itemize}
  \item Doppler range: $\approx -3115$ to $-2916$ Hz.
  \item CRLB RMSE lower bound: $\approx 48$ km (weak due to millisecond timestamp noise).
  \item Single-run MAP error: $\approx 136$ m (synthetic realization only).
\end{itemize}
These numbers are diagnostics for code correctness and algorithm behavior; they
are not claimed as achievable accuracy and should not be cited as such.

\subsection{Required Validation before Paper Claims}
The following are required before any localization accuracy claim in a
publication:
\begin{enumerate}
  \item 100+ Monte Carlo trials with randomized noise and independent geometry.
  \item Reported mean and confidence intervals for localization error.
  \item Comparison of achieved RMSE against CRLB bounds.
  \item Sensitivity analysis across elevation angle, pass duration, and SNR.
  \item HIL run with extracted observations from real IQ captures.
\end{enumerate}
