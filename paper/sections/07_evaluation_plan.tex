\section{Evaluation}\label{sec:evaluation}

All evaluations to date are \textbf{synthetic and trace-driven}.
No real satellite OTA data is used.  Results are reproducible by running the
scripts listed below.  No paper performance claims are made from these runs.

\subsection{Reproducible Synthetic Pass}
The \texttt{generate\_synthetic\_dataset.py} script produces a deterministic
synthetic satellite pass (400~km altitude, 53° inclination, 10-minute duration)
and exports geometry and observations to:
\begin{itemize}
  \item \texttt{experiments/results/synthetic/synthetic\_pass\_dataset.json}
  \item \texttt{experiments/results/synthetic/synthetic\_pass\_observations.csv}
\end{itemize}
The geometry serves as the reference for all subsequent experiments.  Output
includes clean and noisy Doppler and delay series, carrier frequency, drift,
time offset, and noise levels---all recorded in the JSON metadata.

\subsection{Monte Carlo Coarse Localization Study}
The \texttt{run\_monte\_carlo\_synthetic.py} script runs repeated estimator
trials over the fixed synthetic geometry with independent noise realizations.
Each trial produces MAP location, error magnitude, posterior entropy, and HPD
region size.  Results are exported to:
\begin{itemize}
  \item \texttt{experiments/results/montecarlo/monte\_carlo\_trials.csv}
  \item \texttt{experiments/results/montecarlo/monte\_carlo\_summary.json}
\end{itemize}
The summary JSON records mean, median, p90 error magnitude, and posterior
entropy diagnostics.  These are \textbf{preliminary diagnostics only};
the number of trials (configurable via \texttt{--trials}) should be increased
for any published result, and confidence intervals must be reported.

\subsection{CRLB Timestamp Sensitivity Diagnostic}
The \texttt{diagnose\_crlb\_sensitivity.py} script evaluates the classical
CRLB lower bound across a sweep of timestamp noise $\sigma_\tau$.  Results are
exported to:
\begin{itemize}
  \item \texttt{experiments/results/crlb/crlb\_sensitivity.csv}
  \item \texttt{experiments/results/crlb/crlb\_sensitivity\_summary.json}
\end{itemize}
The sweep confirms that at $\sigma_\tau = 1$ ms (range noise $\approx 300$ km)
the CRLB for position exceeds several hundred km, establishing that sub-ms
timestamp accuracy is needed for meaningful range information.  The classical
CRLB assumes unbiased estimation with no priors on nuisance parameters; the
MAP estimator used in this work applies weak priors and is not directly
comparable to the classical CRLB.

\subsection{Doppler-Time Ambiguity and Ablation}
The \texttt{run\\_ambiguity\\_ablation.py} script evaluates estimator behavior
across seven ablation cases that progressively add observation modalities and
nuisance parameters.  Cases cover: single vs. multi-time frequency observations,
with/without delay observations, and with/without CFO and drift nuisance.

Key preliminary observations from synthetic runs:
\begin{itemize}
  \item Multi-sample frequency observations yield lower posterior ambiguity than
        a single snapshot, consistent with the intuition that the Doppler curve
        shape (not just a scalar) carries geometric information.
  \item When CFO and drift are included as nuisance, posterior entropy increases
        slightly, reflecting that these parameters partially absorb geometric
        information during marginalization.
  \item Delay observations help when $\sigma_\tau$ is small; at $\sigma_\tau=1$ ms
        the timing channel adds negligible information (range noise $\approx300$ km).
  \item All results are \textbf{preliminary synthetic diagnostics} and do not
        constitute paper performance claims.
\end{itemize}
