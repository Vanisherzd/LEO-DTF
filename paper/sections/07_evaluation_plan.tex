\section{Evaluation}\label{sec:evaluation}

All evaluations to date are \textbf{synthetic and trace-driven}.
No real satellite OTA data is used.  Results are reproducible by running the
scripts listed below.

\subsection{Reproducible Synthetic Pass}
The \texttt{generate\\_synthetic\\_dataset.py} script produces a deterministic
synthetic satellite pass (400~km altitude, 53° inclination, 10~min duration)
and exports the geometry and observations to
\texttt{experiments/results/synthetic/synthetic\\_pass\\_dataset.json} and
the corresponding CSV.  This is used as the reference geometry for all
subsequent experiments.

\subsection{Monte Carlo Coarse Localization Study}
The \texttt{run\\_monte\\_carlo\\_synthetic.py} script runs repeated estimator
trials over a fixed geometry with randomized noise realizations.  It outputs
per-trial results (MAP location, error magnitude, posterior entropy, HPD region
size) to \texttt{experiments/results/montecarlo/monte\\_carlo\\_trials.csv} and
a summary JSON.

Results are preliminary diagnostics; no paper performance claims are made
from these runs.

\subsection{CRLB Timestamp Sensitivity Diagnostic}
The \texttt{diagnose\\_crlb\\_sensitivity.py} script evaluates the CRLB lower
bound across a sweep of timestamp noise $\sigma_\tau$.  It demonstrates that
at $\sigma_\tau=1$ ms (range noise $\approx300$ km) the CRLB for position is
several hundred km, confirming that sub-millisecond timing is needed for
meaningful range information.  The CRLB is a theoretical lower bound only;
the MAP estimator with priors is not directly comparable to classical CRLB.

\subsection{Trace-driven HIL Plan}
Section~\ref{sec:hil} describes the hardware-in-the-loop validation design.
No HIL runs have been executed; no real IQ captures are used in this draft.
